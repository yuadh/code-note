# 编程思想

**面向过程：**分析出解决问题所需的步骤，然后用函数把这些步骤一一实现，使用的时候再一个一个一次的调用就可以了

**面向对象：**把事务分解成为一个个对象，然后由对象之间分工合作

面向对象的特性：

  • 封装性
  • 继承性
  • 多态性

# 类和对象

# 对象

对象时一个具体的事物

在 JavaScript 中，对象是一组无序的相关属性和方法的集合，所有的事物都是对象

  • 属性：事物的特征，在对象中用属性来表示
  • 方法：事物的行为，在对象中用方法来表示

# super

可以调用父类中的调用函数

class Father{
    constructor(x,y){
        this.x=x;
        this.y=y;
    }
    sum(){
        console.log(this.x+this.y);
    }
}
class Son extends Father{
    constructor(x,y){
        super(x,y);
    }
}
var demo = new Son(4,5);
demo.sum();

### 2022/01/01

> 2021年终总结 and 2022年规划

  首先是梳理下2021年每个时间点做的事

- 2021.01~2021.06 准备专升本
- 2021.06~2021.09 暑假工
- 2021.09~2021.12 大三上

似乎又是没有意义的一年，专升本学习的知识似乎没啥用. 信息技术、毛中特、英语、C语言几乎只是为考试准备而学习而且考试还很水.暑假工也是做的与专业不想关的工作，大部分学习的东西可能在这个暑假都忘光了.而现在就读的专升本也和来之前规划满满的预期不一样.回想下这学期做的事，似乎又浪费了一个学期.嗯..刚换学校适应环境用了段时间，也有一大堆琐事不过都不应该成为没学习的理由。

在编程这方面似乎也没有我想象的这么热爱，在读大学之前总觉得自己是热爱编程的可是三年下来，似乎在编程水品方面也没有很大的提升.三年如果能拿出大部分时间花在编程学习研究上也不会是现在这个水品了.三年只有皮毛的编程水品，毫无竞争力..时间大部分应该都花在游戏、网上冲浪、看片上了

似乎也没必要写这个年终总结了，嘛了大专三年至现在的专升本第一学期即将学期末时间都浪费了..如果能多花点时间在有意义上不会到现在都一事无成..至少不会连个编程项目都没..

but 但是我花钱花时间专升本不是为了继续浪费时间的，至少不能和大专三年一样浪费时间..所以写这篇日级总结下2021年，对2022年做一些规划.虽然对编程没我这么热爱未必要走编程程序员码农这条路，但是似乎我也没有其它的方向.在没有其它方向之前我需要继续学习编程知识，做好本专业的学习尽力当名程序袁...码农我对编程也没有厌恶的地步...然后是2022年规划

2022年规划

结合之前做的不好的地方、没有做到的地方以及没有想到的地方提出以下问题

- 优柔寡断，该做的事情没有果断去做
- 总待在舒适圈，沉迷低级趣味网上冲浪
- 时间管理差，对未来的事过于担忧焦虑反而浪费了时间
- 太自闭，思维行动太僵硬无论在学习还是生活处理上

提出一下建议、解决方案

- 时间管理，活在当下做好目前可以做好的事
- 思想健康积极点，走出自闭
- 开放思维，不要让路走的太吃力
- 量的积累，量的积累到质的飞跃
- 尝试不同的道路，有时候换个环境换个方式会更好找到解决问题的办法

写到现在似乎还没对2022年做出什么很好的具体规划，而具体规划也不在此写出.以上对2021年的总结和对2022年的规划是大体的方向，虽然没有具体的写出细节但足够总结我的2021和开始我的2022新一年.此时再说的什么禁止，什么别在重蹈覆辙都是bullshit，等一段时间后再看此篇日记..在看写它的意义

具体目标不在此列出，以下列出几个大致目标

- [ ] 2022年 GitHub  提交全绿
- [ ] 坚持学习，读博客，背单词
- [ ] 戒300天300，然后找到实习
- [ ] 走出舒适圈，时间管理等...

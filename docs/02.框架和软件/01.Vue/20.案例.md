## props 变量配置项

- type ：数据类型

- required ：是否必须传入参数

- 自定义校验规则

  `validator`

  ```js
  props:{
  arr:{
   type:Array,
   required:true:
  validator(value){
  if(value.length>2 && value.legnth<5){
      return true                  
    }
  }else{
   return  false
  }
  }
  }
  ```

### 处理品牌

**注意**

因为 `vue3` 中的 `setup()` 函数是立即执行的，所以在发送请求时 

`async setup(){await API()}`  会有问题 ，尽量避免使用 `async`

```js
const brand = reactive({
    id:'brand',
    name:'品牌',
    children:[{id:'brand-children',name:'品牌推荐'}],
    brands:[]
})

findBrand().then(data=>{
    brand.brands = data.result
})
```

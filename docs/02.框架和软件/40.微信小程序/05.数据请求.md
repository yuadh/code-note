## 数据请求



需要在小程序的后台管理中配置请求域名

- 域名只支持 https 协议
- 域名不能使用 ip 地址或 localhost
- 域名必须经过 ICP 备案



## 网络请求

### get

```js
Page({
  btnHandle(){
   	wx.request({
      url:'https://www.escook.cn/api/get',
      method:'GET',
      data:{
        name:'zs',
        age:22
      },
      success: (res)=>{ //请求成功之后的回调函数
        console.log(res)
      }
    }) 
  }
})
```



### post

```js
Page({
  btnHandle(){
   	wx.request({
      url:'https://www.escook.cn/api/get',

        age:22

  }
})
```

what is git ?

Git is a version control system ,

It helps you keep track of all the files in a project 

Git tracks every change you make 

if you make a mistake , it can reset a  file to match a previous version 

the collection of all versions of your project files is known as  a Git repoitory

it's basically a folder in which you edit  your files 

then run  Git commands to store your changes and pull in changes from other contributors

Git is used by teams of developers working on a project together

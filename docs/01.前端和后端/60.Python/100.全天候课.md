## 字典

### 创建字典

- {{key,val},}
- dict()

```python
 #-*- coding: UTF-8 -*- 

contacts = {'张艳':13873321234,
'杨丽':13623456712} #创建contacts字典
student = {'Name':'张艳'
,
'Age':20,3:13873321234} #创建student字典
emptydict = {} #创建空字典
dict1 = {1:'空调'
,2:'1.5p'
,2:1989} #创建dict1字典
print(contacts) #输出字典
print(student) #输出字典
print(emptydict,dict1) #输出字典
```

**创建区别**

```python
# -*- coding: UTF-8 -*- 
student = dict((['Name','张艳'],('Age',20),'xy',('d',{1:2})))
contacts = dict([(('tel','email'),[13873321234,'zhangyan@163.com'])])
dict1 = dict(Name='杨丽',Age=18,a=(1,2))
dict2 = dict()
print(student)
print(contacts)
print(dict1,dict2)
```

## 集合

```python
# -*- coding: UTF-8 -*- 
s1 = set('hello') #创建由字符串字母组成集合
s2 = set(('hello',)) #创建一个英文单词集合
print(s1,s2)
s1 = set(['hello','world'])
s2 = set(('hello','Python'))
print(s1,s2)
s1.add('!') #给s1集合添加字符
s1.update(s2) #给s1集合添加s2集合
print(s1,s2)

```

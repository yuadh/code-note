## 题目

串口发送事件处理函数使用按键，且D5灯亮

串口接收使用事件处理使用函数，且D5灯亮

## 关键代码

- 初始化串口、注册串口任务事件、测试串口是否可用

```c
MT_UartInit();
MT_UartRegisterTaskID(task_id);
HalUARTWrite(0,"hello!",6);
```

- 准备发送串口服务函数 和 接收串口服务函数

串口接收服务函数 ： 功能数据回显

```c
static void GenericApp_UartRxMsgs(mtOSALSerialData_t *rxpkt){
  uint8 len,*UartRxBuffer=NULL;
  UartRxBuffer = rxpkt->msg;
  len = *UartRxBuffer;
  HalUARTWrite(0,UartRxBuffer+1,len);
  HalUARTWrite(0,"\r\n",2);
}
```

串口发送服务函数：闪烁D5灯后，发送串口数据

在 `GenericApp_HandleKeys(uint8 shift,uint8 keys)`

```c
if ( shift )
  {
    if ( keys & HAL_KEY_SW_6 )
    {
      // 亮灯
      HalLedBlink(HAL_LED_1,1,HAL_LED_DEFAULT_DUTY_CYCLE, HAL_LED_DEFAULT_FLASH_TIME);
      // 发送串口数据
      HalUARTWrite(0,"hello!",6);
    }
    if ( keys & HAL_KEY_SW_7 )
    {
      // 亮灯
      HalLedBlink(HAL_LED_2,1,HAL_LED_DEFAULT_DUTY_CYCLE, HAL_LED_DEFAULT_FLASH_TIME);
      // 发送串口数据
      HalUARTWrite(0,"hello!",6);
    }
    if ( keys & HAL_KEY_SW_3 )
    {
    }
    if ( keys & HAL_KEY_SW_4 )
    {
    }
  }
```
